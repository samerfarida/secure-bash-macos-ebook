# Part I – Bash Fundamentals

In this first section, you’ll build a strong foundation in Bash scripting specifically tailored for macOS users, administrators, and security engineers.

**What you’ll learn:**

- Why Bash remains essential on macOS, despite modern shells like Zsh
- How to confidently navigate the Terminal and manage your shell environment
- Core Bash syntax, quoting, variables, and command structures
- Practical file system operations: navigating, creating, and securing files and directories
- Essential text processing and data manipulation tools used in real-world macOS workflows

By the end of this part, you’ll have the skills to write clear, reliable Bash scripts that align with macOS’s unique environment and security requirements.
